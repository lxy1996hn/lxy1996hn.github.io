intelligent system: konwledge bases

Posted on 2020-02-11 Edited on 2020-04-23

what is intelligent system

什么是intelligent system，intelligent system 和人类又有什么关系，我们为什么又要学习intelligent system。它能又能为我们做什么呢？

在lecture notes 中老师定义：

clearly defined tasks requiring intelligence and education from human tend to be doable for AI techniques, such as playing chess, performing probability calculation.

但是什么又指intelligence，是指像人类一样能够在复杂环境下做出复杂的选择吗,考虑到计算的复杂性，intelligent system在大部分情况下难以达到人类的智能，所以说是clear defined tasks。 而在introduction to AI 中我们提及到artificial intelligence 是指在machine中创造出类人的智慧，具体表现为能够

achieve (complex) goal in (complex) environment, exploiting information gathered in the environment in order to perform the most appropriate actions, making decision

也就是说我们在的intelligent system 是被要求具备能在复杂环境中能够做出决定，并行动的能力，这种能力要求我们能够接受环境的信息(通过sensor)，整合这些信息。但在实际情况中，intelligent system远远达不到人的水平，比如人可以在人群中一下子认出自己的父母。这也就是为甚么intelligent system现在主要应用于clearly defined tasks。 目前的intelligent system 很难在极其复杂的环境中得出较好的答案。而人类具有良好的adapting/self evaluation .

The history of intelligent system

在50年代到60年代，AI 就已经可以给出简单逻辑结论，和证明一些理论，创造一些计划了。但是遇到一些困难：

ambiguity(highly funded translation programs( Russian to English)were good at syntactic manipulation but bad at disambiguation.

​ “the spirts is willing but the flesh is weak “becomes “The vodka is good but the meat is rotten”

scalability : each are small and can not scale to bigger instances

representation the representation are limited like how to represent the human knowledge??

到了70年代到80年代，更专业的系统被设计了出来，然而overpromising导致了第一次AI winter。

在80年代末以及90年代初，multi-agent system。 deep blue(国际象棋) 让AI back again。 现在AI 取得了一系列的成就deepmind， autonomous car，smart grid。但是是否会有第二个寒冬呢？这谁又能知道呢。

modern intelligent system

现在的智能化系统(自动)有三个层次：

  1. databases(the data are from sensors )

    data collection, extraction(光收集但是不对数据进行分析，只会导致数据爆炸)，integration(形成更高的级别)，querying, indexing(网页), retrieval(比如智慧城市就是一大堆数据点)

  2. data analytics

    data understanding, analytics, visualization, prediction

  3. autonomous systems

    agent learn and adapt, plan with situation awareness; real-time monitoring and actions (无人机，自动驾驶飞机)

logics, knowledge representation (h核心领域)

how do we obtain knowledge

第一问题是怎么定义knowledge？为什么我们需要knowledge？

在AI系统中核心的问题在于

  1. 如何让计算机理解人类已有的知识

  2. 如何让人类理解人工智能新发现的知识

    也就是如何表示知识，让计算机能够理解知识，以及将计算机的知识解码成人类能懂的知识. knowledge representation

    data(singularity)是knowledge representation 中最低级的一员。他是

    a collection of disconnected facts(usually obtained from data)

    比如今天的温度 -15度(温度计测量)

    information(singularity) is 第二层次，information和data区别在information通常由很多的data(linked)组成，能够给我们描述一件稍微复杂一点的事情。但我们很难由information中推测出什么来.

    information is easier to measure, visualize and analyse $\rightarrow$ 对同类信息进行整合

    among facts is established

    比如今天的温度 -15度(温度计测量)和今天是二月10号

    knowledge(generic) 是现在data mining 和AI 的focus，可以得出一定的结论, When we don’t just view information as a description of collected facts, but also understand how to apply it to achieve our goals, we turn it into knowledge. knowledge pyramid

emerge when relationship among patterns are identified and understood

if humidity is high and temperature drops, it is likely to rain.

Different types of knowledge

tacit knowledge : 隐形知识 是指包括经验在内的难以用document 描述处理的。（难以进行逻辑推理的）

explicit knowledge 显性知识包括原理，程序，概念等非常坚固，且能用document 表达(可以进行逻辑推理的，可预测的)（可能抽象？）

factual knowledge 事实知识。 具有时效性，现在有效未来无效。

propositional logics(命题逻辑)

what is proposition logics

proposition logics this is the basic of intelligent system, it is about statement, assertions , sentences that do not have variables(probability). Each statement is either true or false. This is called their true value, which means the unknown things not belongs to proposition logic. 比如，UK 是否会更改和欧盟的贸易协议(unknown)换句话说,在proposition logic里只有发生和不发生

build block: atomic propositions (atomic letter)$\rightarrow$ sentence.

e.g

S1: London is the capital of the UK(build block p) and(operator) the largest city in the UK(build block g)

s2: London is the most expensive place to live in the UK(s) and Bristol is not very far from London(t)

what is propositional logics: syntax, semantics, language

syntax define how a sentence(structurally) is defined.(similar to English grammar)

$\rightarrow$ a set of atomic letters( or proposition). denotes $A={p,q,r,a,b,c}$ that can not broken down into separate propositions. Each atomic proposition is either true or false

$\rightarrow$ a set of operations: negation(not), conjunction(AND)….

e.g ‘Bertrand Russell’ is a proper noun(grammar)

Language is the collection of all possible sentence given A and the set of connectives(what is connectives)(Language is dependent on both set A and the connectives(operations). )???

semantics define the meaning of the sentence, and their interpretations ( is this sentence true or not). 所以对于语义来说判断语句的正确或错误也是他的任务，我们通过TRUTH TABLE

e.g “Bertrand Russel” refers to a British philosopher(meaning)

$ (condition)p ∨ (consequence)q$ true if p and q are both true

$p ∧ q$ true if p or q is true

$p=q$ same value

$p\rightarrow q$ if condition is true then the consequence should be true, if condition is false, then no matter the consequence is, the $p\rightarrow q$ is true.

possible world and semantics

possible world

in order to consider all atomic letters, we define the concept of possible world???. A possible world is an assignment of a truth table to every atomic letter. each possible world can be understood as particular view of the actual world. but only one is the actual world.

$w:A{true,false} A={p,q,r,s}$???

one of the possible world could be

$w(p)=TRUE,W(q)=true…..w(r)=true,w(s)=true$

there are 4 atomic letter, each has two possible value. thus there is 2^4=16 possible world $W={w_1,w_2,w_3….}$

possible world table

express all possible world with its truth table of each atomic letter

e.g

W p q r s

w_1 1 1 0 1

if we need to consider sentences: $p ∧ q$ then we can only focus on some columns

所以说sentence实际上限制了possible world

truth table 和 possible world 的不同

它们的不同在于possible world只是涉及每一个值得正确和错误，而truthtable是在具体的atomic letter 和connectives 下的正确，所以说 truth table 只使用部分atomic letter

relationship between an arbitrary sentence and possible world

model？？？： a possible world w is a model of sentence $\beta$, if $\beta$ is true in this possible world
