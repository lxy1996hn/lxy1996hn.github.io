What is unsupervised learning

aims : Given unlabelled data the aim is to partition the data into distinct cluster of similar element. 首先监督学习是具备特征(feature)和标签(label)的。所谓的学习就是将标签与特征建立联系(mapping)。而无监督学习则没有标签，所以无监督学习没有办法判断正误。

reasons：为什么要采用无监督学习，明明监督学习更有效率，原因在于标签是认为定义的，是非常复杂和需要极大的工作量的。what is unsupervised learning

所以无监督学习的核心是similarity，often based on Euclidean distance(欧式距离) or set distance

Challenge:

  • knowing the optimal number of clusters(K)

  • stability and convergence

similarity and distance

similarity and difference are dual. a suitable measure of difference can be transformed into a corresponding measurement of similarity.

function d is $d : \Omega \rightarrow R+ $ : a distance metric on a set $\Omega$

具备以下特性：

  • reflexive: $\forall x \in \Omega, d(x,x)=0$

  • symmetric: $\forall x,y \in \Omega, d(x,y)=d(y,x)$

  • triangular inequality $\forall x,y,z \in \Omega, d(x,y)\le d(x,z)+d(z,y)$

  • hamming distance??(what is hamming distance)：

    1. Hamming distance is a metric for comparing two binary data strings with equal length.$ \rightarrow$ hamming distance is the number of bit position in which the two bits are different. (就是说在哪个地方不匹配了，那么hamming distance就会在哪个地方为1)

      e.g. 11011001 ⊕ 10011101 = 0100010

如果是一列数$\Omega=R^n,x^\rightarrow=(x_1,x_2,x_3….),y^\rightarrow=(y_1,y_2,y_3….) $

K-means clustering

k means 是最简单的clustering 算法， K 是指的 the number of prototypes (其实是将n个数据对象分为K块)

步骤：

  1. Randomly partition the set of patterns into K sets (将分成K 个set)

  2. for each set of P calculate its mean vector (计算P 个set 的均值，新的中心)

  1. 重新计算距离$||\vec{x}-\vec{x}_p||$,然后根据距离(按远近)重新划分区域

  2. 重复2-3 知道 converge

Hierarchical clustering

层次聚类算法是通过不同类别的相似度来创建有层次的分类

他可以分为top-down 和bottom up 两种

top-down 首先视为一个cluster然后不停的分直到每个cluster里只有一个点

bottom up是将只有一个点的cluster不断向上整合最后只剩一个cluster包含所有的点

Kohonen self organised map？？？

SOM 是一种无监督的人工神经网络，他利用了竞争学习的策略，依靠竞争来优化网络？？？，且用临近关系(neighbourhood function) 来维持空间的拓扑结构？？？(高位数据映射到低位空间)

SOM 的网络结构有两层，输入，输出。

输入层就是外界输入的信息，输出层的个数通常是聚类的个数，代表每一个需要聚成的类(自己定义，或者根据经验类似K)

竞争学习：每个输入的样例在输出层找到和它最匹配的一个节点，陈伟激活节点，接着用随机梯度下降法更新激活节点的参数，同时和激活节点临近远近而适当的更新参数，这种竞争通过神经元之间的横向抑制连接（负反馈）实现。

EM algorithm

EM 算法是一种迭代算法，每一步迭代都分为两步，期望步和极大步。

核心思想是： 首先根据已有的观测数据，估计出模型参数的值，在根据上一步轨迹出来的参数值估计缺失的数据值，在根据缺失的数据值加上已经观测的数据再对参数值进行估计
