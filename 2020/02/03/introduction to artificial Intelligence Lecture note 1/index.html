title：什么是AI

我们虽然经常谈论各种AI，但是什么是AI，AI能够做什么，以及AI的局限性是什么都不知道。
AI 的定义一直都在变化变化

The exciting new effort to make computers think…machine with minds, in the full and literal sense —-(Haugeland 1985)

比较深入人心的定义来自于

artificial intelligence is the discipline concerned with the creation of intelligent behaviour in machines(typically computer system ). By intelligent behaviour we mean the capability in achieve(complex) goal in (complex) environment in order to perform the most appropriate actions,making decision in order to achieve their goal —-(Russell and Norvig)

AI 不再是简单的行为而是复杂的行为，同时需要和环境进行交互，这种交互也是复杂的。AI 需要能够做决定和采取行动。
AI 的目标被主要分为

  1. system that thinks like a man
  2. system that thinks rationally
  3. system that act like a human
  4. system that act rationally

需要注意到的是单纯的复制人类的行为并不是好的AI，因为人类经常做出无意义的事情。engineer skills更适合 think or act rationally

title:我们能用AI 做什么

  1. web page ranking 这是最常用到AI 的地方

  2. Game，能用AI来打游戏，比如自动挂机，象棋游戏中的电脑对手：目前的游戏AI主要集中于AI游戏，AI生成游戏内容，AI 对玩家建模 疯狂的甘地

    • 寻路:网格地图,A-star,navmesh
    • 有限状态机
    • 行为树:NPC行为
    • 模糊状态机
    • 集群控制
  3. autonomous car: 在无人驾驶中很有名的一个问题就是人不能仅仅坐下来监控一个东西，如果人没有参与性，大概率会丧失集中度，这就会导致reaction time变得非常的长。 而reaction time 有可能导致在需要人类接管无人驾驶时，无人车没有办法接管。学界有两个思路，第一个是将人类拉入无人驾驶，让他们通过玩游戏等方式集中精力，另一个就是完全由人工智能托管。google 的waymo采取的就是第二种。事实上，如果有了无人驾驶，人却依然要集中精神驾驶的话，那没有什么意义。
  4. smart Home and healthcare： 比如小米之家，但是现在并没有达到非常智能的境界。

    title：How do we tackle these tasks

  5. 原始的方法：将现实世界进行抽象，形成 formal model，在通过算法来解决问题，最后形成程序. machine learning：是通过training example来训练machine learning形成独特的算法。这可以广泛运用于解决例如字母的矫正 hte 如何自动校正为the。人类不可能编译单独的程序来解决这个问题，但对于machine learning 算法来说，他可以通过各种书写错误最后达到能够校正的程度，machine learning算法并不是完美的。举个例子，machine learning算法没有办法知道你不知道的问题，因为我们并不了解算法的核心，和通过machine learning算法生成的程序。
  6. search 算法。search算法涉及到决策树问题，例如，一盘棋，这么下和那么下都会导致不同的后果。
  7. bayesian network
  8. markov decision process
