machine learning是AI 中很常见的一种学习方法。最早于1959年由Arthur Samuel提出

Field of study that gives computers the ability to learn without being explicitly programmed

也就是说机器学习是一种特殊算法，机器学习的核心在我看来是特征他们是基于特征来的算法，一般包括训练集，特征，建模，模型等步骤，可究竟什么是特征，特征与特征之间又有什么关系，以及训练集与特征集又有什么关系呢在easyAI 中举了一个例子 学习汉字一二三的书写，如何让电脑识别不同人书写的一二三呢。首先是大量的训练集，各种不同人书写的一二三（是否需要包括错误的案例比如4,5,6？？）然后是特征一是一条横线，二是两条横线，三是三条横线。然后需要不停的学习建模就是机器学习。这里有一个问题在于我们如何知道这是有效特征？

Type of machine learning

一般来说，machine learning 分为 4种

  • supervised learning 监督学习

    supervised learning 一般包括两种 classification and regression.

    classification: where$\sigma= {c_1,……,c_n}$ responding to a finite site of class or categories

  • reinforcement learning 强化学习

    regression vs classification

  1. 应用场景不同

    classification常被用来鉴别广告邮件，是离散的：只有是或不是(1 or 0) 而对于regression 来说，更像一种预测，建立线性或非线性关系,是一种逼近关系。例如假设身高和牛奶的关系。假设喝牛奶高，不喝牛奶矮。 classification根据喝牛奶的数量（特征）将数据分为高或矮。而regression就是建立身高与牛奶的具体关系，喝多少牛奶有多高。还可以预测：假设喝50斤牛奶能长多高

    • how to judge performance

    有了数据，并且进行了训练，我们需要知道machine learning 算法的拟合度有多少？ 是否符合原数据

    1. classification

      • what is confusion matrix

        confusion matrix is the combination of predicted and actual values. 假设一个人患了癌症。那么得癌症为1，没得为0. 我们对患癌没患癌进行预测你。

        TP: true positive, the acual cases and the predicted are both 1 (true)

        a person having cancel and the model classified his cases as cancer

TN: true negative, acual cases and the predicted are both 0 (wrong)

 *a person do not have cancer and the model tell he do not have cancer*



FP: false positive: where the acual cases is 0 but the predicted is 1. 



FN: false negative, where the actual cases was 1 but the predicted is 0.  预测的正确还是错误决定了true or false。 预测的1 还是0决定了 negative or positive

   * *what is accuracy and precision* 

     accuracy:  
     $$
     \frac{TP+TN}{TP+TN+FP+FN}
     $$


     precision: not worry unnecessary 
     $$
     \frac{TP}{TP+FP}
     $$


recall:  No missing case. 
$$
precision=\frac{TP}{TP+FN}
$$
  Accuracy vs precision

    accuracy 指的就是实际正确性，我们预测的模型与实际情况有多大的误差。但也可能会出意外

Accuracy should NEVER be used as a measure when the target variable classes in the data are a majority of one class. example

precision：not worry people uncessary.

Precision is a measure that tells us what proportion of patients that we diagnosed as having cancer, actually had cancer. The predicted positives (People predicted as cancerous are TP and FP)

  • what to minimise

错误就是False positive 和 false negative。 但在什么情况下应该偏向于哪种，例如在癌症的例子中，我们会希望尽可能将所有的癌症患者找出来，那么就有可能能提高FP的概率（其实是降低FN概率）。换句话说就会有人没有癌症但是却被诊断为癌症。 再比如邮件，特别是学校的邮件，我们不希望学校的邮件也被认为是sparm 邮件，那么FP的概率就要下降

Generalisation

Generalization is a term used to describe a model’s ability to react to new data. That is, after being trained on a training set, a model can digest new data and make accurate predictions generalisation

在supervised learning中，有一点和其他很不一样的地方在于只有在supervised learning 中overfitting才会成为一个问题。supervised learning 的核心在于数据的训练，基于训练出来后模型的预测。这就意味着越多的数据，可以使得预测的结果更加准确，然而过分的准确会导致数据难以生成。有时候甚至会产生不准确的预测.

太过简单又会导致难以解释variance。如果overfitthing 就会导致太过于specificIt will fail to make accurate predictions with new data because it learned the training data too well.

Train and Test Data

分析生成的模型的好坏需要被分为训练数据和测试数据(注意在数据测试时不能被训练)

直觉上，我们的模型在训练数据集表现的好，在测试数据集上依旧稳定，那么我们就找到了一个好的函数。但是，我们的模型选择并不是这么简单，是一个相对复杂的过程。（因为还有超参数设置，比如学习率）data trainning

经常会有模型在不同情况下表现不一的状况，所以需要对数据进行validation。 就是将训练数据拆分成两部分一部分训练一部分验证测试。验证测试来帮助进行模型选择.

但是这种办法并不是完美的有一个很大问题在于模型和参数的选取极大的依赖于划分方法。 基于此有人提出了交叉论证： repeat many times with different divisions into trainning and test.

即

将数据集分成N份每次取n-1份作为训练集，1份用于validation。然后依次往后推，很明显这样的计算量过大。 于是有了诸如K-fold cross validation 具体见trainning and validation
